AND THE OO PROGRAMMING PARADIGM

Python and the Object-Oriented Programming Philosophy are introduced in the first unit. A programming paradigm based on the idea of object-orientation is called object-oriented programming. The fundamental concept is to match the software architecture to the fundamental realities of the application-specific reality. This unit also includes a Python class on classes and objects. Additionally, there will be a collaborative discussion on the article "The Power of ePortfolio Development" by Di Silvestro & Nadir in addition to a group discussion with students on the subject of "Factors Influencing Reusability" (2021). Python and the idea of object orientation are the main topics of this unit. The key subjects covered were the syntax, object initialization, and how to access object variables.

Develop a Python program and apply protected and unprotected variables within it.

Python Code


# program to illustrate access modifiers of a class

# super class
class Super:
    # public data member
    var1 = None

    # protected data member
    _var2 = None

    # private data member
    __var3 = None

    # constructor
    def __init__(self, var1, var2, var3):
        self.var1 = "Adams"
        self._var2 = 22
        self.__var3 = "AA"

    # public member function
    def displayPublicMembers(self):
        # accessing public data members
        print("Public/unprotected Variable is: ", self.var1)

    # protected member function
    def _displayProtectedMembers(self):
        # accessing protected data members
        print("Protected variable is: ", self._var2)

    # private member function
    def __displayPrivateMembers(self):
        # accessing private data members
        print("Private variable is: ", self.__var3)

    # public member function
    def accessPrivateMembers(self):
        # accessing private member function
        self.__displayPrivateMembers()


# derived class
class Sub(Super):

    # constructor
    def __init__(self, var1, var2, var3):
        Super.__init__(self, var1, var2, var3)

    # public member function
    def accessProtectedMembers(self):
        # accessing protected member functions of super class
        self._displayProtectedMembers()


# creating objects of the derived class
obj = Sub("Geeks", 4, "Geeks !")

# calling public member functions of the class
obj.displayPublicMembers()
obj.accessProtectedMembers()
obj.accessPrivateMembers()

# Object can access protected member
print("The Student's age(which is a protected variable) is:", obj._var2)
