UNIT 5: MORE ON CLASSES

Everything in object-oriented programming is based on classes. As a result, this course will put a greater emphasis on classes and teach students how to assign variables using constructors. Additionally, unit 5 explains Python's modularity as well as the definition and significance of abstract classes. The ideas of inheritance, aggregation, and composition will next be discussed.

Another intriguing idea that was mentioned this week was pair programming. Transferring expertise from experienced programmers to new programmers seems very beneficial. Remote teams can benefit from pair programming because it can be practiced in a remote setting as well.

SKILLS/KNOWLEDGE GAINED

  • In this unit I learnt what abstract classes are all about and why they are important in object oriented programming.
  • I also learnt the various types of polymorphism such as: Dynamic polymorphism, Parametric polymorphism, etc.
  • I learnt the various types of interfaces used in python which are: Formal and Informal.
