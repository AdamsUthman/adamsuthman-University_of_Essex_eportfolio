UNIT 9: PACKAGING AND TESTING

It's important to not ignore code testing. The methods for testing code and the modules that can be used to do so are covered in Unit 9. Furthermore, by describing the PEP257 convention for documenting a Python program, the unit shows how important code documentation is. Additionally, it explores cyclomatic complexity and how object-oriented systems might benefit from it.
